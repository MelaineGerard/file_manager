
# File Manager

File Manager is a simple File Manager (obviously) for Linux.
It's build with Flutter.

## Installation

Install File Manager using Flutter

```bash
git clone https://github.com/MelaineGerard/file_manager.git
cd file_manager
flutter pub get
```
